Quiz Application

ASP .Net MVC 5 and Entity Framework

Created by Naamane Ilias / Basma El Guaabouri / Mehdi Regragui/ Charif Roudali

Notre Application permet de:

  • Ajouter, Modifier, Supprimer un quiz
  • Ajouter, Modifier, Supprimer les questions et reponses d'un quiz
  • Passer un quiz et afficher le score de chaque utilisateur
  • Afficher une alerte rouge si le score inférieur à 8, alerte en jaune si le score entre 8 et 12, en bleu si le score entre 12 et 18 et en vert si supérieur à 18

Technologies used

  • Entity Framework
  • ASP .NET MVC 5
  • AJAX/JQuery
  • Bootstrap/CSS/HTML5

Model Container de l'application

Dossiers utilisés

  • Scripts qui contient les fichier Javascript (Jquery)
  • Views qui contient les Vues du projets
  • Controllers qui lient les données avec les vues
  • Content qui contient les fichiers CSS du projet

Dans cette application on a utilisé 2 controleurs

  • QuizController
  • HomeController

AJAX

La force de notre application consiste à utiiser AJAX et JQuery pour dynamiser les pages Webs

Perspectives du projet

La suite du projet consiste à mettre une application pour la gestion du quiz pour multiutilisateur. C'est à dire qu'un test peut être passé par deux joueurs en temps reel en utilisant une base de donnée Redis (NoSql) et utilisé la connexion client/serveur avec sockets pour la gestion du temps réel.
